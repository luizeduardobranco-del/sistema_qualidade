# Relatório PDF (Estrutura)

Seções (MVP):

1. Identificação
   - ID da RNC, título, cliente, gravidade, status, datas, área/responsável
2. Descrição da ocorrência
3. Investigação
   - Método (5 porquês / Ishikawa / outro)
   - Causa raiz
   - Classificação da causa
4. Plano de Ação (RACP)
   - Lista de ações (tipo, descrição, responsável, prazo, status, custo)
5. Evidências
   - Lista/links de evidências anexadas
6. Conclusão
   - Resumo, validação e encerramento

Saída:

- PDF com layout profissional (capa simples, cabeçalho com logo opcional, rodapé com paginação).
