# Dashboard (KPIs)

KPIs obrigatórios:

- RNC abertas / fechadas (por período)
- % SLA cumprido (por etapa e total)
- Tempo médio de resolução
- RNC por área
- RNC por cliente
- Custo total (RNC + ações, conforme modelagem)

Filtros (MVP):

- Área (multi-seleção)
- Cliente (multi-seleção)
- Gravidade (multi-seleção)
- Período (data início/fim)

Requisitos:

- Combinação livre (AND) entre filtros
- Atualização rápida (cache/consulta eficiente)
